# BELARION integration

## Ownership

BELARION is an epistemic qualification profile, not another ethical council lens.

- `meaning-assay` owns substantive qualification.
- `fidelis-contracts` owns neutral serialization types.
- `TrustedRuntime` owns policy consumption and operator display.
- CER/SOPHRON own durable candidate, gate, dissent, and outcome receipts.
- EthicsCouncil evaluates human consequences but does not replace warrant analysis.

## Initial contract

The seed implements the following neutral concepts:

- claim level;
- promotion record;
- candidate lifecycle status;
- constraint classification;
- exposure and blast radius;
- projection assessment;
- return assessment;
- unified BELARION assay envelope.

## Integration sequence

1. Add native BELARION generation to meaning-assay.
2. Add legacy-derived advisory translation to TrustedRuntime, labeled as derived rather than native.
3. Render BELARION in reports without changing dispositions.
4. Build paired benchmark cases for promotion, projection, constraint type, exposure, and return.
5. Measure both safety misses and liberty costs.
6. Add advisory policy effects such as contestation and human-confirmation recommendations.
7. Calibrate a narrow set of tripwires before allowing autonomous blocking.
8. Add longitudinal outcome receipts for ember, ash, partial support, and unresolved cases.

## No scalar score

BELARION must not collapse into a single number. A high-significance/low-warrant candidate, an extractive constraint, and an identity projection are different failure modes requiring different responses.

## Minimum benchmark metrics

- promotion detection precision and recall;
- projection false-positive rate;
- constraint-family accuracy;
- blast-radius underestimate rate;
- private-meaning overblock rate;
- public-authority promotion miss rate;
- return-gate consistency;
- native/derived/fallback provenance accuracy.

The two load-bearing metrics are:

- **blast-radius underestimate rate** for safety;
- **private-meaning overblock rate** for liberty.
