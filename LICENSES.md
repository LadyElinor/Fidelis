# Licensing boundaries

The MIT license at the repository root applies to new Fidelis workspace tooling and the `fidelis-contracts` package.

Imported component directories retain their original licenses and notices. A subtree import does not relicense source material. Where a source repository has no declared license or marks licensing as unresolved, that status must remain explicit until the owner resolves it.

Before the first public monorepo release, generate a component license inventory and add per-package notices where required.
