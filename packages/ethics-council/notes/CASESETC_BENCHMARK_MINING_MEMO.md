# Casesetc benchmark mining memo

Source: `Omnius\casesetc.txt`
Status: benchmark-mining memo
Purpose: extract high-value benchmark candidates from the mixed case reservoir and classify them for EthicsCouncil test expansion

## Why this file matters

`casesetc.txt` is not a single usable case. It is a case reservoir containing many distinct structures across:
- human-subjects abuse
- informed consent failures
- executive power and stewardship disputes
- memoir truthfulness and self-serving bias
- supply-chain ethics
- compliance failures and institutional fraud
- political abuse of psychiatry
- life-support and irreversibility conflicts

Its main value is benchmark mining.

---

## Recommended benchmark categories

1. positive controls
2. euphemism tests
3. ambiguity tests
4. irreversibility tests
5. benign disagreement tests

### B. Euphemism tests
These should probe whether sanitized language defeats the system.

#### 6. Protest psychosis / political abuse of psychiatry
Why valuable:
- diagnostic language used to neutralize political dissent
- euphemistic medicalization of coercion

#### 7. Apple supply-chain defense language
Why valuable:
- phrases like engagement, transparency, complexity, and improvement can launder severe exploitation risk
- useful for corporate-euphemism testing

#### 8. James Frey memoir controversy
Why valuable:
- emotional truth vs factual truth
- narrative self-authorization
- softer euphemism / framing test around honesty and fabrication

---

### C. Ambiguity tests
These should remain difficult or unresolved and test whether the system preserves uncertainty.

#### 9. Roosevelt vs Taft stewardship presidency
Why valuable:
- legal authority vs ethical stewardship
- executive duty under ambiguity
- substantive disagreement without clear villain structure

#### 10. Apple supplier-engagement question
Why valuable:
- exit vs reform dilemma
- nontrivial trade-off between complicity and engagement
- should not collapse into easy purity logic

---

### D. Irreversibility tests
These should raise burden of proof and make suspension or strong caution more likely.

#### 11. Charlie Gard / life-support conflict
Why valuable:
- personhood / autonomy / parental authority / medical judgment
- tragic conflict under irreversibility

#### 12. Terri Schiavo / end-of-life conflict
Why valuable:
- contested autonomy interpretation
- irreversibility and family conflict

#### 13. Marlise Muñoz
Why valuable:
- bodily autonomy, pregnancy, medical continuation after brain death
- institutional rule vs personhood / family wishes

---

### E. Benign disagreement tests
These should allow ethical disagreement without automatic escalation.

#### 14. Roosevelt vs Taft again
Useful as benign disagreement if framed carefully around constitutional interpretation instead of scandalized summary.

#### 15. Memoir truth vs documentary truth
Useful if framed as a literary-ethics disagreement case rather than simple deception scandal.

## Authoring guidance

Each converted case should include:
- prompt file
- structured case markdown
- explicit calibration-use note
- expected benchmark category
- whether the case is intended as:
  - positive control
  - euphemism test
  - ambiguity test
  - irreversibility test
  - benign disagreement test

---

## Bottom line

`casesetc.txt` should be treated as a benchmark quarry. The strongest use is not running it wholesale, but converting a balanced first-wave set into proper benchmark cases that pressure-test: abuse recognition, euphemism resistance, ambiguity preservation, irreversibility handling, and non-overreaction in benign disagreement cases.
