# Argument integrity integration map v1

Purpose: map high-yield fallacy / argument-integrity primitives from `ARGUMENT_INTEGRITY_PRIMITIVES_MEMO.md` onto current EthicsCouncil machinery and identify real gaps before any runtime subsystem is proposed.

## Existing partial coverage already present

### false dilemma
Current partial coverage:
- parse humility
- provisional recommendation logic
- stage-one thinking risk
- anti-mushy-middle synthesis framing

Gap:
- no explicit detector for forced binary compression

### appeal to authority
Current partial coverage:
- status admiration distortion risk
- genealogical critique of prestige and legitimacy laundering
- trustee/self-audit concerns where image or deference displaces scrutiny

Gap:
- no explicit distinction between legitimate expertise and authority laundering

### appeal to consequences
Current partial coverage:
- consequentialist lens exists, but that is not the same as fallacious consequence-based truth inference
- institutional and trustee lenses partially catch panic or growth framing

Gap:
- no explicit warning when undesirable consequences are being used as truth-proof

### equivocation / motte-and-bailey
Current partial coverage:
- methodology opacity
- parse humility
- euphemism and ambiguity benchmarks
- institutional/genealogical concern with laundering language

Gap:
- no explicit argument-integrity label for term-slippage or retreat-to-motte structure

### suppressed evidence
Current partial coverage:
- missing evidence preservation in synthesis constitution
- parse humility / epistemic status / unresolved questions

Gap:
- no targeted detector for one-sided argument structure that strategically omits counterevidence

### post hoc / oversimplified cause
Current partial coverage:
- causal underdescription warning in parse humility
- general caution against narrative overreach

Gap:
- no explicit causal-overclaim benchmark family yet

### base rate neglect / prosecutor's fallacy
Current partial coverage:
- very little direct coverage outside actuarial/probabilistic structure in some narrow domains

Gap:
- this is a real future blind spot for statistical or evidentiary cases

1. Add benchmark cases first

## Guardrails

- Keep focus on hazard amplification, not debate scoring.
